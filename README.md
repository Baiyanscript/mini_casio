# mini_casio — 多功能数学计算工具箱

> 为 Xiaomi Vela 打造的精简数学工具箱

mini_casio 是一个专为 **Xiaomi Vela**（小米 Vela 系统）设计的轻量级数学计算应用，提供基础计算、方程求解、函数绘图、不等式求解等多种数学工具。界面紧凑，交互流畅，适合在手表、手环等小屏幕设备上使用。

---

## ✨ 功能特色

| 功能模块 | 说明 |
|---------|------|
| **基础计算器** | 支持加减乘除、括号运算、小数计算等基本操作 |
| **方程求解** | 支持一元一次、一元二次方程求解 |
| **函数绘图** | 快速绘制数学函数图像 |
| **不等式求解** | 支持一元不等式计算与解集展示 |
| **分数转换** | 支持小数与分数的相互转换 |
| **高级运算** | 支持 sin、cos、tan、^、√、log 等高级数学函数 |
| **智能验证** | 自动校验数学表达式安全性，过滤危险字符 |
| **友好交互** | Toast 提示、确认对话框、路由导航，体验流畅 |
